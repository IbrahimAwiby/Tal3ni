GET Get All Users
/api/users

Retrieve all users with pagination support.

Query Parameters:

  • page - Page number (default: 1)
  • limit - Number of users per page (default: 100)

Example Response:

{ "success": true, "data": [ { "_id": "657a1b2c3d4e5f6a7b8c9d0e", "username": "Ahmed Mohamed", "phoneNumber": "01012345678", "birthDate": "1990-05-15T00:00:00.000Z", "gender": "male", "carNumber": "123456", "carType": "Toyota Corolla", "createdAt": "2023-12-11T10:30:00.000Z", "updatedAt": "2023-12-11T10:30:00.000Z" } ], "pagination": { "page": 1, "limit": 100, "total": 1, "pages": 1 } }
POST Create User
/api/users

Create a new user with comprehensive validation.

Request Headers:

Content-Type: application/json

Request Body:

{ "username": "string (2-50 characters)", "phoneNumber": "string (Egyptian format)", "birthDate": "ISO Date (past date)", "gender": "male|female", "carNumber": "string (1-8 digits)", "carType": "string (2-50 characters)" }

Validation Rules:

  • ✅ Username: 2-50 characters, required
  • ✅ Phone Number: Egyptian format (010, 011, 012, 015), unique, required
  • ✅ Birth Date: Must be in the past, required
  • ✅ Gender: "male" or "female", required
  • ✅ Car Number: 1-8 digits only, unique, required
  • ✅ Car Type: 2-50 characters, required

Success Response:

{ "success": true, "message": "User created successfully", "data": { "_id": "657a1b2c3d4e5f6a7b8c9d0e", "username": "Ahmed Mohamed", "phoneNumber": "01012345678", "birthDate": "1990-05-15T00:00:00.000Z", "gender": "male", "carNumber": "123456", "carType": "Toyota Corolla", "createdAt": "2023-12-11T10:30:00.000Z", "updatedAt": "2023-12-11T10:30:00.000Z" } }

PATCH Update User
/api/users/:id

Update an existing user. All fields are optional.

Request Body (example):
